# Morella Beauty Store

![Morella Beauty Store](images/morella-store.png)

An online beauty store offering a curated selection of skincare, makeup, and wellness products. Built to deliver a clean, fast, and playful shopping experience.

## About

Morella Beauty Store is an eCommerce website where customers can browse products, add items to their cart, and check out smoothly. The goal is a simple, modern shopping experience with a fun, feminine aesthetic that works well on both desktop and mobile.

## Visit the Store

Live store: **[https://morella-store-2.myshopify.com/](https://morella-store-2.myshopify.com/)**

The store is currently password protected. Password: `000`

## Features

- Browse products by category (Lips, Body, Hair)
- New Arrivals section
- Product detail pages with images and descriptions
- Product videos ("See me in action")
- Customer reviews
- Newsletter signup ("Join the Glow Club")
- Shopping cart and checkout
- Responsive design for mobile and desktop

## Tech Stack

- Shopify (custom theme)
- Liquid templating
- HTML5
- CSS3
- JavaScript
